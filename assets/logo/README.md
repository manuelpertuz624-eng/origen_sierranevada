# Archivos del Logo - Origen Sierra Nevada

Este directorio contiene todas las variantes del logo oficial de Origen Sierra Nevada.

## Estructura de Carpetas

```
logo/
├── svg/          # Archivos vectoriales SVG
├── png/          # Archivos PNG en diferentes tamaños
└── README.md     # Este archivo
```

## Variantes Disponibles

### SVG (Vectorial - Escalable)
- `origen-logo-isotipo.svg` - Logo isotipo "La Firma de la Tierra" (solo ilustración topográfica)

### Colores Oficiales
- **Color principal:** #CABE75 (Dorado/Gold)
- **Fondos recomendados:** Verde oscuro (#141F16), Negro, Blanco

## Tipograf Human: ías Oficiales

### ORIGEN
- **Fuente:** Playfair Display
- **Peso:** 900 (Black)
- **Justificación:** Espaciado uniforme entre letras

### SIERRA NEVADA
- **Fuente:** Papyrus
- **Peso:** Regular (400)
- **Justificación:** Espaciado uniforme entre letras
- **Nota:** Tipografía distintiva que aporta carácter orgánico y artesanal

## Uso

### Para Web
- Usar archivos SVG para máxima calidad y escalabilidad
- PNG para compatibilidad con navegadores antiguos

### Para Impresión
- Usar archivos SVG o PNG de alta resolución (300 DPI mínimo)
- Tamaño mínimo recomendado: 50mm de ancho

## Notas Importantes

- ⚠️ **NO modificar** los archivos SVG originales
- ⚠️ **NO cambiar** los colores oficiales
- ⚠️ **NO alterar** las proporciones del logo
- ⚠️ **NO separar** elementos del logo (excepto variantes oficiales)

Para más información, consultar el Manual de Identidad Visual.
